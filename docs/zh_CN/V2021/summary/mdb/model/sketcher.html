Sketcher#

Sketcher commands are used to define the entities, such as the geometry, constraints, and dimensions, to create a sketch and to store the values and attributes associated with a particular sketch.

Create constrained sketches#

class SketchModel(name, description='', stefanBoltzmann=None, absoluteZero=None, waveFormulation=abaqusConstants.NOT_SET, modelType=abaqusConstants.STANDARD_EXPLICIT, universalGas=None, copyConstraints=ON, copyConnectors=ON, copyInteractions=ON)[源代码]

Abaqus creates a Model object named Model-1 when a session is started.

备注

This object can be accessed by:

mdb.models[name]

备注

Check SketchModel on help.3ds.com/2021.
